== Description ==

Set a primary category for your (custom) posts and query them in your template using native WordPress queries.

== Installation ==

1. Upload the `wp-primary-category` folder to your `/wp-content/plugins/` directory
2. Activate the plugin via the Plugins menu in WordPress
3. A new Primary Category menu will appear in WordPress with options and general help
